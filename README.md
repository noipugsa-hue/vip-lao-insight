# VIP Lao Insight 🎰

แอปพลิเคชันวิเคราะห์และทำนายหวยด้วย AI พร้อมฟีเจอร์ทำนายฝัน

## ✨ Features

- 🏠 **หน้าหลัก**: แสดงเลขเด่น Hot Numbers พร้อม Confidence Score
- ✏️ **ใส่เลขเอง**: ระบบวิเคราะห์เลขที่คุณใส่
- 💭 **ทำนายฝัน**: วิเคราะห์ฝันและให้เลขโชคดี 2, 3, 4 ตัว
- 🏆 **วิน5รวม**: วิเคราะห์แบบพิเศษ
- 🎯 **00-99**: ช่วงเลขทั้งหมด
- 🔍 **วิเคราะห์**: เครื่องมือวิเคราะห์เพิ่มเติม
- 📊 **ความแม่น**: ติดตามความแม่นยำของการทำนาย

## 🚀 Setup

Make sure to install dependencies:

```bash
npm install
# or
yarn install
```

## 💻 Development Server

Start the development server on `http://localhost:3000`:

```bash
npm run dev
# or
yarn dev
```

## 🏗️ Production

Build the application for production:

```bash
npm run build
```

Locally preview production build:

```bash
npm run preview
```
